1) 시작하기 (필수 준비)

필수: 학교에서 안내한 Mahara 사이트 주소, 본인 계정(또는 발급 정보), 과제 안내문.
  • 브라우저는 최신 버전(크롬/엣지/사파리 권장)
  • 스크린샷 저장 폴더: homepage/images/ 에 넣고 문서에 링크

2) 계정 만들기 & 로그인

  1. 학교가 제공한 Mahara 주소 접속
  2. 회원가입(또는 학교 계정으로 로그인)
  3. 프로필에서 이름, 학번(또는 ID), 연락 이메일 확인/수정
주의: 비밀번호 분실 대비, 복구 이메일 정확히 입력.

3) 포트폴리오(페이지) 만들기

  1. 새 페이지 만들기 클릭 → 제목을 ‘나의 학습 포트폴리오’처럼 명확히
  2. 블록 추가(텍스트, 이미지, 파일, 외부 링크 등)로 섹션 구성
  3. 추천 섹션:
    • 학습 성과(과목/프로젝트별 핵심 요약)
    • 성찰(강점·개선점·다음 목표)
    • 증빙자료(보고서/슬라이드/깃허브/영상 링크)
  4. 저장 후 페이지 보기로 최종 확인

텍스트 블록 팁

문단은 3~5문장, 굵게/리스트를 활용해 가독성↑

이미지 블록 팁

캡션을 달아 맥락 설명(날짜/역할/도구)

파일 블록 팁

파일명 규칙: 과목_항목_이름_연도.pdf

4) 제출/공유 방법

  1. 과목 공지의 제출 방식 확인(예: 비공개 링크/그룹 제출/기관 제출 등)
  2. 공유 설정에서 대상과 기간(마감일 포함) 지정
  3. 제출 후 열람 권한 테스트(로그아웃/다른 브라우저로 열어보기)
TIP: 평가자에게 보이는지 미리보기로 최종 확인하세요.

5) 점검 체크리스트

  • 표지(이름/학번/연도)
  • 학습 성과 섹션 3개 이상
  • 성찰 문단(강점/개선/다음 목표)
  • 증빙 링크(최소 3개)
  • 공유 권한/기간 확인

6) 자주 묻는 질문

페이지가 비공개라 평가자가 못 본대요

공유 설정에서 대상(교원/그룹)과 기간을 다시 지정하고 저장하세요.

파일 업로드 용량이 부족해요

용량이 큰 영상은 YouTube(비공개/미등록) 링크로, 문서는 PDF로 최적화.

모바일에서 레이아웃이 깨져요

이미지 가로폭을 100%로 업로드하고, 너무 큰 표는 PDF 링크로 대체.

7) 문제 해결

  • 로그인/접근 오류: 브라우저 캐시 삭제 후 재시도, 다른 브라우저로 확인
  • 이미지 안 보임: 파일 경로/이름(대소문자) 확인, images/ 폴더에 존재?
  • 제출 확인: 로그아웃 상태에서 공유 링크 열어보기
